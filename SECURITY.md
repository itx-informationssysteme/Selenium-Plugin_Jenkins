# Security Policy

## Reporting a Vulnerability

If you find a vulnerability in this plugin, please report it according to the [Jenkins Security Policy](https://www.jenkins.io/security/reporting/).

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them to the Jenkins Security Team at:
- Email: `security@jenkins.io`
- [Jenkins Security Reporting Guidelines](https://www.jenkins.io/security/reporting/)

The Jenkins Security Team will coordinate the fix and disclosure process.

## Supported Versions

Security updates are provided for the latest released version of this plugin.
